# Tools

Tools are how agents interact with the world. A tool is a function that an agent can call to perform actions, retrieve data, or manipulate systems.

In this chapter:

- [Defining Tools](./defining.md) — Create custom tools with Zod schemas
- [Tool Validation](./validation.md) — Advanced validation patterns
- [Preset Tools](./presets.md) — Use built-in tools for common operations
